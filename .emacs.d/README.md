### Emacs Packages and Shortcuts Documentation
#### Shortcuts
##### Emacs Default
###### See Current File Path - Find Alternative File
`C-x C-v`

##### Fuzzy Search
###### Open Files
`C-c C-f`

###### Open Directories
`C-c C-d`

###### Clear Fuzzy Search Cache
`M-x fiplr-clear-cache`

##### Hide and Show Blocks of Code (Collapse)

###### Show Block
`C-c @ C-s`

###### Hide Block
`C-c @ C-h`

###### Toggle Hide/Show
`C-c @ C-c`

###### Show All
`C-c @ C-M-s`

###### Hide All
`C-c @ C-M-h`
